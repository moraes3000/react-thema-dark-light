---

## Como baixar o projeto

```bash 
# Clonar o respositório
$git clone git@github.com:moraes3000/react-thema-dark-light.git

# Entrar no respositório
cd react-thema-dark-light

# Instalar as dependências
$yarn install

# Iniciar o projeto
$yarn start 
```
